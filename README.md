## Deploy att
Compile and deploy ATT.sol, call its generateTokens() function

## Installation
~~~
cd att_token
npm install
~~~

## Run
~~~
node testATT.js
~~~
ps. before you run testATT.js, you need to configure a ethereum private chain according to https://github.com/cryptape/ethereum-bootstrap
